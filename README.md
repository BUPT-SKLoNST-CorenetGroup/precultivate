
## 1. 仓库介绍

本项目用于北京邮电大学-计算机学院-网络与交换技术国家重点实验室-交换与智能控制研究中心-王尚广老师团队-核心网研究方向，实习生、研0学生进组培训。

通过多个任务，培养学生分析问题、查询资料、解决问题、表达的能力，增加对计算机基础知识和核心网有关知识的掌握，锻炼相关工具的使用。


## 2. 仓库使用

仓库中会按照类别，罗列多个任务，不同类型的学生可以根据自己的情况选择不同的任务完成。任务的提交，可以参考[任务0](https://github.com/zxm-bupt/precultivate/blob/main/task-0/%E4%BB%BB%E5%8A%A1%E5%88%9D%E5%A7%8B%E5%8C%96%E4%B8%8E%E8%80%83%E6%A0%B8%E6%96%B9%E5%BC%8F.md)。

每个任务分为三部分，任务概述、任务描述、需要掌握的内容。

任务概述部分会大致介绍本任务相关的知识点、参考资料和使用到的工具。对于使用到的工具，本项目并不强制使用，使用者可以选择自己喜欢的工具链，只要能够完成任务描述中的步骤就可以。

任务描述部分，给出的任务不会是详细的实验步骤，更像是产品经理提出的需求，我们会尽量让该需求明确，但是使用者也需要尽可能理解需求，并且根据需求调研出实现的步骤，并完成。

需要掌握的内容是完成任务的过程中，需要掌握的基础知识，我们不希望使用者死记硬背，而是对相应的知识点有相应的理解。知识点中标有*号的知识点，为重要知识点，或者说通识，需要使用者能够不假思索的给出自己的理解。其余知识点，只需要知道该知识点的基本内容，以及如何查询即可。

我们不排斥使用AI，反而鼓励借助AI来解决问题。但是不鼓励copy AI的想法和代码。我们希望，人是AI的使用者，而不是调用AI的工具。
